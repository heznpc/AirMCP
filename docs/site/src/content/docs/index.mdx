## What is AirMCP?

AirMCP is an Apple-native [Model Context Protocol](https://modelcontextprotocol.io/) server for macOS. It connects Claude, Codex, Cursor, and other MCP clients to native Apple applications through JXA (JavaScript for Automation), a Swift bridge, and direct system APIs. AirMCP is the governed connector and control layer; the client does the reasoning.

## Quick Start

```bash
npx airmcp init
```

The interactive setup wizard will guide you through module selection and MCP client configuration. See the [Installation guide](/getting-started/installation/) for detailed instructions.
