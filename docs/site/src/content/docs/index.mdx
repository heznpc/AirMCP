## What is AirMCP?

AirMCP is a governed [Model Context Protocol](https://modelcontextprotocol.io/) runtime for the Apple ecosystem. It connects Claude, Codex, Cursor, and other MCP clients to Apple applications through JXA (JavaScript for Automation), a native Swift bridge, AppIntents, and direct system APIs. AirMCP is the connector and control layer; the client does the reasoning.

## Across Apple Platforms

The macOS runtime is available today, and the native iOS/iPadOS runtime is in preview. visionOS is a roadmap target for spatial interaction and native actions, while watchOS is a roadmap companion for commands, notifications, and per-call approvals through a paired iPhone. Mac-only automation routes to a paired Mac runtime.

## Quick Start

```bash
npx airmcp init
```

The interactive setup wizard will guide you through module selection and MCP client configuration. See the [Installation guide](/getting-started/installation/) for detailed instructions.
